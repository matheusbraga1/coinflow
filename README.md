# CoinFlow

Personal finance tracker built with .NET 10, Clean Architecture and Domain-Driven Design.

## Status

🚧 In active development.

## Tech stack

- .NET 8
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- MediatR
- JWT authentication

## Architecture

Clean Architecture with two bounded contexts:
- **Lançamentos** (core monolith): users, accounts, categories, transactions, budgets
